# PLP — testid contract

| Key                       | data-testid                   | Element                                                            |
|---------------------------|-------------------------------|--------------------------------------------------------------------|
| `plpRoot`                 | `plp-root`                    | PLP page root container                                            |
| `plpBreadcrumb`           | `plp-breadcrumb`              | Breadcrumb row                                                     |
| `plpBreadcrumbItem`       | `plp-breadcrumb-item`         | Individual breadcrumb link                                         |
| `plpTitle`                | `plp-title`                   | Category page title heading                                        |
| `plpSortBar`              | `plp-sort-bar`                | Sort bar container                                                 |
| `plpSortOption(slug)`     | `plp-sort-option-<slug>`      | A single sort option (cheapest / newest / …)                       |
| `plpActiveSort`           | `plp-active-sort`             | Currently selected sort chip                                       |
| `plpFilterSidebar`        | `plp-filter-sidebar`          | Filter sidebar container                                           |
| `plpFilterGroup(slug)`    | `plp-filter-group-<slug>`     | A collapsible filter group (price، brand، color، …)                |
| `plpFilterToggle(slug)`   | `plp-filter-toggle-<slug>`    | A specific filter toggle inside a group                            |
| `plpFilterClear`          | `plp-filter-clear`            | «حذف فیلترها» button                                              |
| `plpAvailableOnlyToggle`  | `plp-available-only-toggle`   | «فقط کالاهای موجود» toggle                                         |
| `plpGrid`                 | `plp-grid`                    | Product grid                                                       |
| `plpCard`                 | `plp-card`                    | A single product card                                              |
| `plpCardTitle`            | `plp-card-title`              | Title text inside a product card                                   |
| `plpPagination`           | `plp-pagination`              | Pagination control container                                       |
| `plpNextPage`             | `plp-next-page`               | «صفحه بعد» button                                                 |
| `plpEmptyState`           | `plp-empty-state`             | Empty-state container («نتیجه‌ای یافت نشد»)                         |

Fallback rules: same as `../homepage/testid-mapping.md`.
